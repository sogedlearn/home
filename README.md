# SOGED

An innovative digital learning platform designed to provide accessible, interactive, and engaging educational experiences for students, educators, and institutions.

🌐 Website: https://sogedlearn.com

---

## About SOGED Learn

SOGED Learn is an educational web platform developed to support online learning through digital resources, interactive courses, and modern technologies.

The platform aims to provide:

- Online courses
- Educational resources
- User authentication
- Multilingual support
- Responsive design
- Scalable architecture
- Future academic tracking capabilities

---

## Project Overview

SOGED Learn was conceived as a modular educational ecosystem capable of growing over time and incorporating new functionalities according to institutional and student needs.

The project focuses on:

- Accessibility
- User experience
- Educational innovation
- Maintainability
- Scalability

## Development Process

### Phase 1 – Planning and Design

The project started with the definition of its educational objectives and user experience strategy.

Special attention was given to:

- Interface simplicity
- Accessibility
- Mobile responsiveness
- Intuitive navigation

### Phase 2 – Frontend Development

The user interface was built using HTML, CSS, and JavaScript, following a modular approach that facilitates future expansion.

This stage included:

- Landing pages
- Course pages
- Reusable components
- Responsive layouts
- Language support

### Phase 3 – Authentication System

An authentication system is currently being integrated using Supabase.

Features include:

- User registration
- Secure login
- Session handling
- Password recovery
- Role-based access (planned)

### Phase 4 – Educational Content

The platform structure supports the integration of:

- Digital courses
- Downloadable materials
- Interactive learning experiences
- Student progress tracking
- Certification systems

### Phase 5 – Deployment

SOGED Learn is hosted through GitHub infrastructure and configured with a custom domain.

Website:

https://sogedlearn.com

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Soged-Project.git
```

Navigate to the project folder

```bash
cd Soged-Project
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm start
```

or

```bash
node server.js
```
